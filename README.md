# DSA Learning Code

This repository contains Data Structures and Algorithms (DSA) learning code implementations in Java.

## Contents

### Dynamic Programming
- **FibonacciDP.java** - Fibonacci sequence using dynamic programming
- **CanSum.java** - Can sum problem with memoization
- **Mincoin.java** - Minimum coin change problem
- **DP coin 1.java** - Coin change dynamic programming variant 1
- **DP coin 2.java** - Coin change dynamic programming variant 2
- **Subset_sum.java** - Subset sum problem

### Knapsack Problem
- **Knapsack.java** - 0/1 Knapsack problem implementation
- **FractionalKnapsack.java** - Fractional knapsack greedy approach
- **Knapsackoops.java** - Knapsack with object-oriented design

### Linked Lists
- **LinkedList.java** - Basic linked list implementation
- **DoubleyLinkedeList.java** - Doubly linked list implementation
- **connect2LL.java** - Connect two linked lists


   ```bash
   javac FileName.java
   ```

   ```bash
   java FileName
   ```
